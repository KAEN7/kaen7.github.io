## 배열

``` javascript
let 배열  = ['사과', '자몽', '포도', '메론'];

배열[0] // '사과'
배열[2] // '포도'
```

## 구구단

Math.random()
소숫점으로 랜덤한 숫자값을 꺼내옴

Math.floor()
가장 큰 정수만 반환함

Math.floor(Math.random() * 9) + 1;
하면 1부터 9까지 랜덤한 숫자를 가져오게됨
